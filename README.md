# numpy_creator

## 项目简介
`numpy_creator` 是一个用于创建和操作NumPy数组的Python可视化程序。该程序提供了一些便捷的方法来生成、修改和分析NumPy数组。

## 安装说明
1. 克隆此仓库到本地：
    ```bash
    git clone https://github.com/YangyuTRUE/npyviewer2.0.git
    ```
2. 进入项目目录：
    ```bash
    cd npyviewer2.0
    ```
3. 安装依赖项：
    ```bash
    pip install -r requirements.txt
    ```

## 使用方法
```python
python numpy_creator 
```
